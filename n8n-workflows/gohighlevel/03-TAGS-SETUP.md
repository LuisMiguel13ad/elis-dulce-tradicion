# Step 3: Create Tags for Customer Segmentation

Tags help you segment customers for targeted marketing and automation triggers.

---

## 3.1 Navigate to Tags

1. Go to **Settings** (gear icon)
2. Click **Tags**
3. You'll see any existing tags listed

## 3.7 Tag Naming Convention

**Important:** Keep tag names lowercase with hyphens. This makes them:
- Easier to reference in automations
- Consistent across systems
- Less prone to typos

Good: `birthday-cake`
Bad: `Birthday Cake` or `BirthdayCake`
